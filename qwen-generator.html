🔍 Record Types

  • vocabulary → Standard words, phrases, or UI terms. Supports multi-character sequences.
  • radical → Structural hanzi components. Always use the dictionary pinyin of the source character (e.g., 水 → shuǐ), not the descriptive name.
  • bonus-character → Standalone characters for reinforcement or etymology. Treated as single-character drills.

📏 Core Rules

  • Header must be exact: Question/English Meaning,Chinese Characters,Pinyin (in parentheses),Type,Notes
  • Pinyin wrapped in (), syllables space-separated. Length must match character count.
  • One entry per line. Leave Notes empty if unused.
  • Wrap fields containing commas in double quotes: "Word, phrase",汉字,(pīn yīn),type,hint

All processing happens locally in your browser. No data is sent anywhere.
